# TUX-ENGINER

Tux enginer es una aplicacion creada con la finalidad de apoyar a los nuevos usuarios de linux a que puedan usar de una manera mas eficiente este sistema operativo.

# Justificacion del proyecto

la razon principal del porque se esta creando esta aplicacion es por que yo [MENDOZAINC] he notado que muchas personas que estan en busca de un S.O Como lo es linux puedan entender como funciona este sistema sin muchas complicacions, ademas de acompanarlos en la instalacion de muchas herramientas y aplicaciones que se encuentran en este S.O ya que muchos no encuantran una forma correcta de instalar muchas dependencias, app o herramientas y terminan regrensando a sus sistemas originales por falta de ayuda en el uso de este S.O

Otra de las razones es por que no tenia nada que hacer xd y se me ocurrio la idea en vacaciones, ya que tenia dudas de como hacer aplicaciones de escritorio con GTK ya que habia usado QT en python pero queria algo un poco mas estandarizado para aplicaciones en linux.
